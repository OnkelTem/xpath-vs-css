# DOM querying performance: XPath vs CSS selectors

A very simple demo, showing the difference in performance of selecting DOM nodes between using [`document.querySelectorAll()`](https://developer.mozilla.org/en-US/docs/Web/API/Document/querySelectorAll) and [`document.evaluate()`](https://developer.mozilla.org/en-US/docs/Web/API/Document/evaluate).

Demo page: https://xpath-vs-css.vercel.app/
